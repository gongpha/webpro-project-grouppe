!!! อ่านก่อนเข้าใช้เว็บ !!!
-------------------
เว็บไซต์ของเราจำเป็นต้องใช้ Module PHP ที่มีชื่อ GD
ซึ่ง XAMPP ได้ทำการติดตั้งไว้ให้แล้ว
แต่จะต้องทำการเปิดขึ้นมาก่อน

1. ไปที่ php.ini (XAMPP Control Panel > Config ตรง Apache > PHP (php.ini) )
2. ค้นหาบรรทัดที่มีคำว่า ";extension=gd"
3. เมื่อเจอ ";extension=gd" แล้ว ให้ทำการลบ Semicolon ออก บรรทัดนั้นก็จะกลายเป็น "extension=gd"
4. บันทึกไฟล์ และรีสตาร์ท Apache
